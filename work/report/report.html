2) Explain the role of HTML, CSS and JavaScript in web development

HTML is what the web page is going to look like. It has the components of the page inside of it and it is a skeleton for the webpage. CSS is how the page is going to look like, it is a styling sheet for the HTML components. So that you can connect your HTML file to the CSS file and style your components. JS allows you to have executable code in the webpage. Its used to create responsive, interactive elements for web pages, enhancing the user experience.

3) Give a brief explanation of how the browser, the HTTP protocol and the web server interacts.

The browser complies the code and it uses HTTP protocols to interact with the web server. in oder words, the browser request a resource from the server, and the server responses back with the resource, and this communication is done over HTTP protocol.
